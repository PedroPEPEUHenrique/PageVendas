Capítulo 07 Express.js

JWT — JSON Web Token

JWT é um padrão aberto (RFC 7519) para transmissão segura de informações em formato compacto. Amplamente utilizado em autenticação stateless.

// Geração do token
import jwt from 'jsonwebtoken';

const token = jwt.sign(
  { id: usuario.id },
  process.env.JWT_SECRET,
  { expiresIn: '1h' }
);
